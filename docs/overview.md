# Jungle Bell - 프로젝트 개요

## 무엇인가?

**Jungle Bell**은 크래프톤 정글 수강생을 위한 macOS/Windows 시스템 트레이 앱이다.
정글 LMS(출석 시스템)를 백그라운드에서 모니터링하고, 학습 시작/종료 체크인 시간이 되면 트레이 아이콘 색상과 카운트다운으로 알려준다.

> **중요:** 이 앱은 체크인을 **자동으로 수행하지 않는다.** 버튼 클릭이나 폼 제출은 사용자가 직접 한다. 앱은 오직 상태를 모니터링하고 알림만 준다.

비공식 커뮤니티 도구 (크래프톤과 무관).

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 상태 감지 | LMS 페이지의 출석 테이블과 버튼을 파싱해 현재 상태 파악 |
| 아이콘 색상 | 흰색(정상) / 빨간색(행동 필요) / 주황색(로그인 필요) |
| 카운트다운 | 다음 상태 전환까지 남은 시간 표시 (트레이 툴팁) |
| 출석 페이지 열기 | 트레이 메뉴 → LMS 페이지를 별도 윈도우로 오픈 |
| 자동 일일 리셋 | 매일 04:00 KST에 체크인 상태 초기화 |
| 세션 유지 | 주기적으로 WebView 리로드 (세션 만료 방지) |

## 기본 스케줄 (KST)

```
04:00 ~ 10:00  →  학습 시작 체크인 창
10:00 ~ 23:00  →  학습 진행 중
23:00 ~ 04:00  →  학습 종료 체크아웃 창
```
