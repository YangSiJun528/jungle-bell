## 테스트 전략

### 테스트 레벨 구분

| 레벨 | 범위 | 상태 |
|------|------|------|
| **단위 테스트** | 문구 검증, phase 계산 | 기존 82개 + 보강 가능 |
| **통합 테스트** | `compute_tick()` → `TickResult` action 검증 (mocking 불필요, 순수 함수) | 미구현, 향후 추가 |
| **수동 QA** | OS 네이티브 알림 발송, 트레이 렌더링, 외부 링크 열기, 설정 창 UI | 아래 체크리스트 |
